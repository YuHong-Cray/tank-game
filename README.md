# 小坦克大战 🎮

一个 **Battle City** 风格的坦克对战游戏，纯 HTML5 Canvas 实现。

## 🎯 游戏特色

- **玩家控制**：方向键移动，空格射击
- **敌方 AI**：自动追踪、射击、智能碰撞躲避
- **大本营系统**：5点HP，防护墙，护盾特效
- **国旗主题**：玩家 🇺🇸 vs 敌方 🇮🇷
- **粒子特效**：爆炸、火花、护盾闪光

## 🎮 操作方式

| 按键 | 功能 |
|------|------|
| ⬆️ ⬇️ ⬅️ ➡️ | 移动坦克 |
| 空格键 | 射击 |
| R 键 | 重新开始 |
| P 键 | 暂停/继续 |
| M 键 | 静音切换 |

## 🏗️ 项目结构

```
tank-game.html     ← 主游戏文件（所有代码都在一个文件中）
CRAY.md            ← 项目上下文文档
```

## 🚀 运行方式

直接在浏览器中打开 `tank-game.html` 即可游玩！

## 📜 技术栈

- **纯 HTML5 Canvas** — 无任何外部依赖
- **JavaScript** — 原生 ES6 面向对象
- **Canvas 2D API** — 所有图形渲染
- **CSS3** — 布局与样式
